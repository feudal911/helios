HELIOS

Sistema de Monitoramento e Manutenção Preditiva de Fazendas Solares

Plataforma completa para gerenciamento de fazendas solares, oferecendo monitoramento em tempo real, manutenção preditiva e alertas inteligentes.

Ver no GitHub Documentação

Recursos Principais

Monitoramento em Tempo Real

Acompanhamento de geração de energia, eficiência e status das placas solares em tempo real.

Dashboard Interativo

Visualizações e gráficos dinâmicos com filtros avançados por período, parque e inversor.

Manutenção Preditiva

Alertas inteligentes para manutenção preventiva com regras personalizáveis.

Geolocalização

Integração com mapas para cadastro e visualização geográfica de placas solares.

Conversão de Energia

Sistema de conversão kW para reais com comparação de empresas compradoras.

Gerenciamento Completo

CRUD completo para parques, inversores, placas solares e regras de alerta.

Tecnologias

  • Python 3.11
  • Flask (Framework Web)
  • MySQL (Banco de Dados)
  • SQLAlchemy (ORM)
  • Chart.js (Gráficos)
  • Leaflet.js (Mapas)
  • Bootstrap 5 (UI)
  • JavaScript (Puro)

Como Usar

  1. Clone o repositório:
    git clone https://github.com/feudal911/helios.git
  2. Instale as dependências:
    pip install -r requirements.txt
  3. Configure o banco de dados MySQL
  4. Execute a aplicação:
    python app.py
